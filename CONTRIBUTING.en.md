# Contributing To JUSTIS

JUSTIS welcomes criticism, cases, sources, translations, and structural improvements.

The goal is not to protect the current wording of the Code. The goal is to make the procedure for discovering justice stronger.

## Best Contributions

The most useful contributions are:

- strong objections to current laws;
- difficult moral or legal cases;
- hidden assumptions in the project;
- corrections to translations;
- new authoritative sources;
- clearer argument trees;
- better procedural safeguards.

## Before You Contribute

Read:

1. `README.en.md`
2. `INDEX.en.md`
3. `codex/UNIVERSAL_JUSTICE_CODE.md`
4. `codex/translations/UNIVERSAL_JUSTICE_CODE.en.md`
5. `support/codex/JUSTIFICATION_INDEX.md`

## Contribution Types

### Moral Question

Add or propose a new question for `OPEN_QUESTIONS.md`.

### Objection

State the strongest objection you can find. A good objection should name the law, principle, protocol, or assumption it challenges.

### Case

Submit a real or hypothetical case that tests the Code. The best cases create tension between two plausible principles.

### Source

Add a source only when it helps clarify an argument, objection, tradition, or concept.

### Translation

Translations should preserve meaning before style. If a term is uncertain, mark the uncertainty instead of hiding it.

## Decision Rule

No contribution becomes part of the Code merely because it was submitted.

For a principle to enter the Code, it must be traceable through:

- a question;
- a protocol;
- arguments and objections;
- a decision;
- a Code update.

## Tone

Be precise, charitable, and adversarial toward weak reasoning rather than toward people.
