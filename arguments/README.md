# Дерева аргументів

У цій папці зберігаються структуровані дерева аргументів і контраргументів для аксіом, законів Кодексу справедливості й конфліктних тем.

## Призначення

Фінальний Кодекс справедливості має бути лаконічним. Але кожна коротка теза повинна мати прозорий шлях обґрунтування.

Саме для цього існує папка `arguments`.

## Підпапки

- `laws` - дерева аргументів для законів Кодексу.
- `axioms` - дерева аргументів для базових аксіом.
- `topics` - дерева аргументів для конфліктних тем.

## Правило

Жоден короткий закон Кодексу не повинен залишатися без:

- протоколу дискусії;
- дерева аргументів;
- умов перегляду.
