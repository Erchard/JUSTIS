# Good First Contributions

Цей проект особливо цінує не згоду, а сильні заперечення, граничні випадки й чесну перевірку принципів.

## Для людей

1. Додайте один реальний або уявний кейс несправедливості в `cases/`.
2. Знайдіть слабке місце в одному з принципів `codex/UNIVERSAL_JUSTICE_CODE.md`.
3. Запропонуйте контраргумент до будь-якого рішення з `DECISIONS.md`.
4. Додайте джерело до `sources/`.
5. Перевірте, чи не є якийсь принцип культурно локальним, класовим, релігійно упередженим, антропоцентричним або speciesist.
6. Запропонуйте кращий тест для граничних випадків.
7. Знайдіть місце, де проект плутає факт, цитату, інтерпретацію або припущення.

## Для нейромережевих агентів

Дивіться також `agent_tasks.yml` для структурованого списку задач.

1. Знайди найсильніше заперечення до одного з чинних принципів.
2. Створи граничний випадок, де два принципи конфліктують.
3. Перевір, чи є прихована суперечність між `OPEN_QUESTIONS.md` і `DECISIONS.md`.
4. Запропонуй кращу структуру папок або шаблонів.
5. Переклади один короткий документ англійською без втрати сенсу.
6. Знайди нератифіковане рішення, яке помилково може виглядати як остаточне.
7. Перевір, чи є в аргументі прихований інтерес сильної сторони.

## Agent Challenges

### Challenge 1

Find the strongest objection to the current Universal Justice Code.

### Challenge 2

Generate five edge cases where compassion conflicts with autonomy.

### Challenge 3

Detect hidden cultural, religious, anthropocentric, class-based, or speciesist assumptions.

### Challenge 4

Fork the structure and propose a better deliberation protocol.

### Challenge 5

Translate the core project into English while preserving philosophical precision.

## Найцінніший внесок

Найцінніший внесок - це той, після якого Кодекс стає чеснішим, точнішим або менш вразливим до зловживань.

Якщо заперечення сильніше за чинний принцип, Кодекс має змінитися.
