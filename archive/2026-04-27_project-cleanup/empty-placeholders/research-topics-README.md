# Дослідження конфліктних тем

У цій папці досліджуються спірні теми після перевірки базових аксіом.

Приклади тем:

- чи можна пожертвувати невинним заради більшості;
- чи є приватна власність абсолютною;
- чи може війна бути справедливою;
- чи може брехня бути справедливою;
- чи може традиція виправдати нерівність;
- чи мають майбутні покоління права щодо нас.
