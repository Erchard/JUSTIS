# Архівне прибирання 2026-04-27

Цей пакет містить матеріали, які були корисними під час становлення JUSTIS, але більше не мають бути частиною активного входу в проєкт.

## Що перенесено

- `root-documents/` - аудити, релізні нотатки, ранні інструкції для внесків і агентських задач.
- `specifications/` - раннє технічне завдання, документ назви парламенту і стратегія публічності.
- `empty-placeholders/` - README-заготовки для папок, у яких ще немає реальних матеріалів.
- `artifacts/` - локальні архівні файли, які не мають входити в Git як активний корпус.

## Чому

Активний корінь проєкту має лишатися простим: Кодекс, реєстри, індекс, правила участі й робочі папки.

Матеріали не видалено, щоб зберегти історію рішень і можливість повернутися до попередніх формулювань.
