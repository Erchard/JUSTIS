# Архів

У цій папці зберігаються застарілі або замінені матеріали.

Файл варто переносити сюди, якщо:

- його зміст замінено новішим документом;
- він більше не є робочим;
- його потрібно зберегти для історії рішень.

Не видаляємо старі матеріали без потреби: краще архівувати і пояснювати причину.
