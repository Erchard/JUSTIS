# Кодекс

Ця папка містить текст Кодексу справедливості та службові індекси до нього.

Головний файл:

- `UNIVERSAL_JUSTICE_CODE.md` - чистий перелік законів Кодексу без статусів, службових приміток і обґрунтувань.

Службові файли:

- `STATUS_INDEX.md` - статуси законів і пов'язані рішення.
- `JUSTIFICATION_INDEX.md` - посилання на протоколи, дерева аргументів і дослідження.
- `LAWS.md` - розгорнуті записи законів із JML-блоками.
- `TERMINOLOGY.md` - різниця між аксіомою, принципом і законом Кодексу.

Правило папки: фінальний Кодекс має бути лаконічним, а доказова база має зберігатися окремо.
