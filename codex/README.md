# Кодекс

У цій папці зберігатимуться тексти самого Універсального кодексу справедливості.

Поточні основні файли:

- `UNIVERSAL_JUSTICE_CODE.md` - короткий Кодекс;
- `LAWS.md` - розгорнуті записи законів із JML-блоками.

Також тут можуть зберігатися:

- коротка декларація;
- презумпції;
- правила зважування;
- словник понять.

Принцип потрапляє сюди лише після проходження дослідницького циклу і фіксації рішення в протоколах.
