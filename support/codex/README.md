# Робоча Підтримка Кодексу

Ця папка містить допоміжні матеріали до Кодексу. Вони навмисно винесені з `codex/`, щоб папка `codex/` містила тільки кінцевий продукт і переклади.

## Файли

- `support/codex/STATUS_INDEX.md` - статуси законів і пов'язані рішення.
- `support/codex/JUSTIFICATION_INDEX.md` - посилання на протоколи, дерева аргументів і дослідження.
- `support/codex/LAWS.md` - розгорнуті записи законів із JML-блоками.
- `support/codex/TERMINOLOGY.md` - різниця між аксіомою, принципом і законом Кодексу.
- `support/codex/TRANSLATION_POLICY.md` - правила канонічної латини й офіційних перекладів.

Правило: `codex/` є вітриною, `support/codex/` є майстернею.
