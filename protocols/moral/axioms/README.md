# Протоколи базових аксіом

Тут зберігаються повні дискусійні протоколи базових питань Кодексу.

Поточні завершені аксіоми:

- Q-001: моральне значення страждання;
- Q-002: гідність людини;
- Q-003: однакові випадки і послідовність;
- Q-004: сила і право;
- Q-005: моральна вага слабшого;
- Q-006: правда і справедливість (`protocol_done`, ще не `codex_done`).

Поточний змістовний крок:

- перевірити Q-006 на готовність до `codex_candidate` або явно позначити його як `revisendum` / `aporia`.
