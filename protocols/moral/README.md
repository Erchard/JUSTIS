# Моральні протоколи

Ця папка містить протоколи, що відповідають на питання про зміст справедливості.

## Підпапки

- `axioms` - базові принципи й аксіоми Кодексу.
- `topics` - прикладні конфліктні теми.
- `cases` - повні протоколи кейс-тестів, якщо кейс змінює або перевіряє зміст Кодексу.

Моральний протокол може вести до нового закону, презумпції, правила зважування або апорії в Кодексі.
