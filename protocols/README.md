# Протоколи дискусій і рішень

## Призначення папки

У цій папці зберігаються протоколи дискусій Небесного парламенту, прийняті рішення, статуси консенсусу, відкриті заперечення і підстави для майбутнього перегляду.

Ця папка відрізняється від `specifications`:

- `specifications` містить методологію, джерела, технічне завдання і загальні карти проекту;
- `protocols` містить конкретні дискусії, рішення і протоколи голосу авторитетів.

## Що сюди зберігати

- протоколи обговорення назв, методів, аксіом і принципів;
- рішення Небесного парламенту;
- статуси результатів: повний консенсус, обмежений консенсус, презумпція, апорія, правило зважування;
- ключові аргументи сторін;
- контраргументи;
- умови перегляду рішення;
- посилання на пов'язані специфікації.

## Рекомендований формат назви файлів

```text
YYYY-MM-DD_short-topic.md
```

Приклади:

```text
2026-04-26_heavenly-parliament-name.md
2026-04-26_methodology-second-reading.md
2026-04-26_axioms-initial-review.md
```

## Статус папки

Папка створена як робоче місце для майбутніх протоколів. Наявні рішення, які вже були зафіксовані в `specifications`, можуть бути пізніше перенесені або продубльовані тут у форматі протоколів.
