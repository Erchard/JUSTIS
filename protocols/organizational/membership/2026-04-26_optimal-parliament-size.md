# Протокол: оптимальна кількість членів Небесного парламенту

## Дата

2026-04-26

## Тема

Чи є поточні 41 індивідуальний член Небесного парламенту оптимальною кількістю для роботи, чи склад потрібно структурувати інакше?

## Питання

Чи є 41 член парламенту забагато, замало або нормально?

Як організувати склад так, щоб:

- не втратити широту традицій;
- не перетворити кожну дискусію на надто довгу;
- не допустити диктатури більшості;
- не виключити незручні голоси;
- зберегти практичну керованість сесій.

## Контекст

Поточний склад:

> 41 індивідуальний член.

Проблема:

- для повного символічного парламенту 41 член може бути нормально;
- для кожної окремої сесії 41 активний голос може бути забагато;
- не кожен автор однаково релевантний до кожної теми;
- деякі голоси потрібні постійно, інші - лише тематично.

## Позиції Небесного парламенту

### Арістотель

Арістотель пропонує дивитися на функцію.

Його позиція:

- оптимальна кількість залежить від мети;
- для кожної теми потрібні ті, хто має релевантну чесноту розсудливості;
- занадто велика рада втрачає практичну керованість;
- занадто мала рада втрачає пропорційність і широту.

### Габермас

Габермас наголошує на умовах дискурсу.

Його позиція:

- важливо не число саме по собі, а чи представлені всі релевантні голоси;
- дискусія має бути керованою;
- потрібно розрізняти постійне ядро і тематично запрошених учасників.

### Ролз

Ролз пропонує модель репрезентативного ядра.

Його позиція:

- кожна сесія має включати представників ключових позицій;
- не всі 41 мають говорити щоразу;
- важливо, щоб позиції були представлені чесно, навіть якщо не всі автори беруть участь.

### Амартія Сен

Сен підтримує гнучкість.

Його позиція:

- краще мати широкий резерв, але невелику робочу групу на сесію;
- залежно від теми склад має змінюватися;
- практична дискусія не повинна чекати, доки висловляться всі 41.

### Ніцше

Ніцше застерігає від середньої думки.

Його позиція:

- велика кількість учасників може породити безпечний середній консенсус;
- іноді один радикальний критик корисніший за десять помірних голосів;
- кількість не повинна гасити силу одиничного заперечення.

## Рішення

Небесний парламент ухвалює:

> 41 індивідуальний член є прийнятною кількістю для повного складу парламенту, але не для обов'язкової участі в кожній сесії.

Запроваджується багаторівнева модель складу.

## Модель складу

### 1. Повний склад

Усі індивідуальні члени парламенту.

Поточна кількість:

> 41

Роль:

- зберігати повний корпус авторитетів;
- підтримувати широту традицій;
- бути резервом для різних тем.

### 2. Робоче ядро

Оптимальна кількість:

> 9-13 членів

Роль:

- брати участь у більшості ключових сесій;
- представляти базові осі справедливості;
- забезпечувати стабільність методології.

### 3. Тематична колегія

Оптимальна кількість:

> 5-9 членів

Роль:

- розглядати конкретну тему;
- включати найрелевантніших авторів;
- не дублювати голоси без потреби.

### 4. Критичний опонент

Кожна важлива сесія має мати принаймні одного сильного критика консенсусу.

Роль:

- шукати приховані припущення;
- атакувати слабкі місця;
- не давати дискусії перетворитися на згоду за інерцією.

### 5. Дорадчі традиції

Традиції й школи не рахуються як індивідуальні члени, але можуть бути залучені як перспективи.

## Рекомендована кількість для сесії

Для звичайної сесії:

> 7-11 активних голосів.

Для великої методологічної сесії:

> 11-15 активних голосів.

Для короткого організаційного питання:

> 3-5 активних голосів достатньо, якщо питання не зачіпає базову методологію.

Для фундаментальної аксіоми:

> 11-15 активних голосів, включно з критиками й різними традиціями.

## Мінімальне представництво для фундаментальної теми

Для фундаментальних питань бажано мати:

- деонтологічний голос;
- наслідковий або практичний голос;
- чеснотний або арістотелівський голос;
- ліберальний голос;
- комунітарний голос;
- правовий голос;
- релігійно-етичний голос;
- глобально-етичний або міжкультурний голос;
- критик прихованих мотивів;
- голос вразливого або стражденного;
- голос майбутніх поколінь або відповідальності.

## Статус

Повний процедурний консенсус.

Поточна кількість 41 не змінюється, але вводиться правило:

> Не всі члени парламенту беруть участь у кожній сесії. Для кожної сесії формується релевантна колегія.

## Умови перегляду

Рішення потрібно переглянути, якщо:

- повний склад перевищить 60 індивідуальних членів;
- робочі сесії стануть надто повільними;
- певні голоси системно не залучаються;
- тематичні колегії стануть надто вузькими;
- буде втрачено міжкультурний баланс.
