# Авторитетні джерела для проекту

## Проект: Універсальний кодекс справедливості

## 1. Призначення документа

Цей документ фіксує корпус авторитетних джерел, які варто дослідити під час створення "Універсального кодексу справедливості".

Мета корпусу джерел - не зібрати максимально велику бібліографію, а визначити тексти, традиції й авторів, які допомагають відповісти на головне питання проекту:

> Які принципи справедливості можуть бути універсальними і стояти вище законодавства окремих держав, приватних інтересів, культурних звичаїв та історичних випадковостей?

## 2. Принципи добору джерел

Джерела добираються за такими критеріями:

- історична впливовість;
- глибина розуміння справедливості;
- здатність формулювати принципи, а не лише правила;
- наявність універсального або міжкультурного виміру;
- придатність для критики несправедливого закону;
- значення для сучасної етики, права і політичної філософії;
- можливість практичного застосування до реальних конфліктів.

## 3. Перший пріоритетний корпус

Ці джерела варто вивчити першими, бо вони формують основну вісь проекту.

### 3.1. Іммануїл Кант - "Основи метафізики моралі"

Ключова роль:

- людська гідність;
- людина як ціль, а не лише засіб;
- категоричний імператив;
- універсалізація морального правила.

Для проекту особливо важливо:

- сформулювати принцип гідності;
- розробити тест універсалізації;
- відокремити моральний принцип від приватної вигоди.

### 3.2. Джон Ролз - "Теорія справедливості"

Ключова роль:

- справедливість як чесність;
- завіса незнання;
- початкова позиція;
- принципи справедливого суспільного устрою;
- захист найменш забезпечених.

Для проекту особливо важливо:

- розробити тест завіси незнання;
- сформулювати принцип справедливого розподілу;
- перевіряти правила з позиції тих, хто не знає свого майбутнього статусу.

### 3.3. Амартія Сен - "Ідея справедливості"

Ключова роль:

- порівняльний підхід до справедливості;
- зменшення реальної несправедливості;
- критика надмірної зосередженості на ідеальних інституціях;
- увага до людських можливостей і фактичного життя.

Для проекту особливо важливо:

- не перетворити кодекс на абстрактну утопію;
- оцінювати не лише формальні правила, а й реальні наслідки;
- розвивати практичні тести справедливості.

### 3.4. Майкл Сендел - "Справедливість. Що є правильним робити?"

Ключова роль:

- порівняння утилітаризму, лібералізму, кантівської етики і арістотелівської традиції;
- питання спільного блага;
- критика моральної нейтральності;
- зв'язок справедливості з чеснотою і громадянством.

Для проекту особливо важливо:

- побачити конфлікти між різними моделями справедливості;
- не зводити справедливість лише до свободи вибору або користі;
- розробити мову для обговорення спільного блага.

### 3.5. Ганс Кюнг - "Проект світового етосу" і "Глобальна відповідальність"

Ключова роль:

- глобальна етика;
- міжрелігійний моральний консенсус;
- золоте правило;
- ненасильство, правдивість, справедливість, партнерство;
- відповідальність людства за мир.

Для проекту особливо важливо:

- знайти мінімальний спільний моральний фундамент між культурами;
- сформулювати універсальні принципи без створення нової релігії або ідеології;
- поєднати справедливість із глобальною відповідальністю.

### 3.6. Арістотель - "Нікомахова етика" і "Політика"

Ключова роль:

- справедливість як чеснота;
- розподільча і виправна справедливість;
- пропорційність;
- мета людського життя і політичної спільноти;
- зв'язок права, чесноти і блага.

Для проекту особливо важливо:

### 3.7. Тома Аквінський - розділи про закон і справедливість у "Сумі теології"

Ключова роль:

- вічний закон;
- природний закон;
- людський закон;
- зв'язок закону і морального порядку;

Для проекту особливо важливо:

- сформулювати критерії, за якими закон може втрачати моральну силу;

### 3.8. Загальна декларація прав людини

Ключова роль:

- сучасний світський морально-правовий консенсус;
- гідність людини;
- рівність прав;
- свобода совісті, слова, безпеки, участі в суспільному житті;
- захист від тиранії і дискримінації.

Для проекту особливо важливо:

- порівняти наші принципи з найвпливовішим міжнародним документом про права людини;
- перевірити, які принципи вже мають глобальну правову мову;
- виявити межі декларативного підходу.

### 3.9. Декларація світового етосу, Парламент світових релігій, 1993

Ключова роль:

- спроба сформулювати спільну етику світових релігій;
- людяність;
- золоте правило;
- ненасильство;
- справедливий економічний порядок;
- правдивість;
- рівність і партнерство чоловіка та жінки.

Для проекту особливо важливо:

- порівняти релігійно-етичний консенсус із філософськими теоріями справедливості;
- використати модель мінімального спільного морального ядра;
- уникнути конфесійної замкненості.

### 3.10. Ганс Йонас - "Принцип відповідальності"

Ключова роль:

- відповідальність перед майбутніми поколіннями;
- етика технологічної цивілізації;

Для проекту особливо важливо:

- застосувати справедливість до екології, технологій і штучного інтелекту.

- EU AI Act як матеріал для аналізу ризиків, відповідальності і контролю.

Теми для дослідження:

- людський контроль;
- недискримінація;
- прозорість;
- відповідальність;
- безпека;
- вплив на працю, владу, приватність і майбутні покоління.

Можлива користь для кодексу:

- застосувати принципи справедливості до нових технологій;
- розробити критерії відповідальної автоматизації;
- врахувати ризики невидимої влади алгоритмів.

## 12. Порядок дослідження

Рекомендований порядок роботи:

1. Кант - для принципів гідності та універсалізації.
2. Ролз - для завіси незнання і справедливого устрою.
3. Сен - для практичного порівняння справедливості.
4. Сендел - для карти головних підходів і конфліктів.
5. Кюнг - для глобальної етики і міжкультурного ядра.
6. Арістотель - для пропорційності, чесноти і розподілу.
7. Аквінський - для природного права і критики несправедливого закону.
8. Загальна декларація прав людини - для сучасної правової мови гідності.
9. Декларація світового етосу - для міжрелігійного консенсусу.
10. Йонас - для відповідальності перед майбутнім.
