# v0.1.0 - First Public Deliberation Corpus

Status: prepared release notes.

## Summary

This is the first public corpus version of JUSTIS: an open, forkable, agent-readable project for building a Universal Code of Justice through structured deliberation, objections, edge cases, and revisable decisions.

JUSTIS is not a manifesto of final truth. It is a public procedure for testing claims about justice.

## Included

- project structure for specifications, protocols, research, arguments, cases, sources, templates, archive, and codex;
- initial Universal Justice Code materials;
- open questions and decision registers;
- procedural integrity standards;
- Heavenly Parliament protocols;
- Justice Markup Language materials;
- case testing framework;
- GitHub issue forms for questions, objections, cases, sources, structural proposals, and agent contributions;
- pull request template;
- `llms.txt` for agent-readable entry;
- `agent_tasks.yml` for structured agent work;
- `CITATION.cff` for citation metadata;
- `PROJECT_BRIEF.en.md` as an English entry point.

## Recommended release title

`v0.1.0 - First Public Deliberation Corpus`

## Recommended public description

JUSTIS is an open, agent-readable project for building a Universal Code of Justice through structured deliberation, objections, edge cases, and ratified moral principles.

It is not a manifesto of final truth, but a public procedure for testing claims about justice.
