# Дослідження

Ця папка містить робочі дослідження аксіом, конфліктних тем і граничних випадків.

## Підпапки

- `axioms` - перевірка базових аксіом проекту.
- `topics` - дослідження конкретних спірних тем.

## Правило

Дослідження не є фінальним законом кодексу. Воно готує матеріал для протоколу рішення і подальшого внесення до `codex`.
