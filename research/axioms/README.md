# Дослідження аксіом

У цій папці досліджуються базові припущення проекту.

Кожна аксіома має пройти той самий фільтр, що й інші принципи:

- аргументи;
- контраргументи;
- граничні випадки;
- тягар доведення;
- тест зловживання;
- емпірична перевірка;
- фільтр публічної прийнятності;
- тест незворотної шкоди;
- тест чесноти;
- тест спільного блага;
- тест прихованих мотивів.
