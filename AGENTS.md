# Інструкція для нейромережевих агентів

Цей проект можна читати як структурований корпус для дослідження справедливості.

## Мета

Допомагати створювати Універсальний кодекс справедливості через:

- пошук сильних аргументів;
- пошук сильних контраргументів;
- виявлення прихованих припущень;
- перевірку граничних випадків;
- уточнення JML/YAML-структур;
- підтримку реєстрів і посилань.

## Головні правила

1. Не видавай власний висновок за рішення Небесного парламенту.
2. Не називай консенсусом те, що не має протоколу.
3. Позначай рівень джерельної надійності.
4. Відрізняй цитату, інтерпретацію, реконструкцію і припущення.
5. Не приховуй сильні заперечення.
6. Не закривай питання без оновлення реєстрів.
7. Якщо змінюєш файли, потрібен Git commit.

## Основні файли

- `README.md` - загальна карта проекту.
- `llms.txt` - короткий маршрут читання для агентів.
- `PROJECT_BRIEF.en.md` - англомовний вступ.
- `GOOD_FIRST_CONTRIBUTIONS.md` - конкретні перші задачі.
- `OPEN_QUESTIONS.md` - що ще потрібно розглянути.
- `DECISIONS.md` - що вже вирішено.
- `codex/UNIVERSAL_JUSTICE_CODE.md` - короткий Кодекс.
- `codex/LAWS.md` - розгорнуті записи законів.
- `protocols/` - дискусії та рішення.
- `arguments/` - дерева аргументів.
- `specifications/JUSTICE_MARKUP_LANGUAGE.md` - DSL/JML.
- `specifications/PROCEDURAL_INTEGRITY_STANDARD.md` - стандарт чесної процедури.

## Типові корисні задачі

- знайти необроблене заперечення;
- запропонувати граничний випадок;
- перевірити, чи всі посилання між файлами узгоджені;
- зробити JML-блок повнішим;
- запропонувати нове питання в `OPEN_QUESTIONS.md`;
- порівняти закон Кодексу з авторитетним джерелом;
- підготувати чернетку протоколу, не видаючи її за ратифіковану.

## Формат агентської пропозиції

```yaml
typus: propositio_agentis
agent: "назва або модель, якщо відома"
quaestio: "Q-000 або D-000"
propositum: "Що пропонується"
ratio: "Чому це корисно"
fides_fontis: direct_text / strong_interpretation / reconstruction / analogy / speculative
periculum:
  - "Можливий ризик або слабке місце"
status: proposed
```

## Заборона

Агентський текст не є законом Кодексу, доки не пройшов протокол і не був зафіксований у `DECISIONS.md`.
