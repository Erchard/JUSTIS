# Джерело: Іммануїл Кант - Основи метафізики моралі

## 1. Бібліографічні дані

Автор: Іммануїл Кант.

Твір: `Grundlegung zur Metaphysik der Sitten` / "Основи метафізики моралі".

Рік першої публікації: 1785.

## 2. Короткий зміст

Твір формулює деонтологічну основу моралі: моральна дія має випливати не лише з наслідків або схильностей, а з принципу, який може мати загальну обов'язковість.

Для JUSTIS найважливіший мотив: особу не можна розглядати лише як засіб для чужої мети.

## 3. Ключові ідеї

- Людина має моральний статус, який не зводиться до користі.
- Моральне правило має бути здатним до універсалізації.
- Особа не повинна використовуватися лише як інструмент.
- Гідність відрізняється від ціни: те, що має гідність, не має простого еквівалента.

## 4. Значення для проекту

Кант є ключовим джерелом для Q-002 і майбутнього Q-007.

Його позиція підтримує Закон 2:

> Людину не можна зводити до користі, ресурсу, ролі або засобу для чужої мети.

## 5. Принципи, які можна витягнути

- Заборона повної інструменталізації особи.
- Універсалізаційний тест морального правила.
- Межа проти арифметики більшості, якщо вона стирає статус особи.

## 6. Заперечення або ризики

- Ризик надто раціоналістичної бази гідності.
- Потрібно не виключити дітей, людей з важкими порушеннями, хворих або залежних.
- Потрібно врахувати матеріальні умови гідності, яких у Канта недостатньо.

## 7. Пов'язані конфліктні теми

- Q-002: гідність людини.
- Q-007: універсалізація морального правила.
- Q-011: жертва невинного заради більшості.
- Q-018: автономія над тілом і життям.

## 8. Потенційні цитати для перевірки

- Формула людства як цілі самої по собі.
- Відмінність між ціною і гідністю.
- Формула універсального закону.
