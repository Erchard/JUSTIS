# Джерельні нотатки

У цій папці зберігаються нотатки по джерелах, авторах і традиціях.

Кожен файл має допомагати відповісти:

- які ключові ідеї дає джерело;
- які принципи можна з нього витягнути;
- які заперечення воно створює проти нашої методології або кодексу;
- які цитати варто перевірити й використати пізніше.

Для нового джерела використовуйте шаблон:

`templates/source-note.md`
