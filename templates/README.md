# Шаблони

Ця папка містить робочі шаблони для повторюваних документів проекту.

Шаблони можна копіювати як основу для нових:

- повних протоколів дискусій: `deliberation-protocol.md`;
- коротких чорнових резюме без статусу повної дискусії: `protocol.md`;
- досліджень аксіом;
- досліджень конфліктних тем;
- нотаток по джерелах;
- законів кодексу;
- дерев аргументів;
- JML-записів;
- сесій Небесного парламенту;
- оцінки членів Небесного парламенту;
- структурованих виступів і мовленнєвих актів;
- тестових кейсів.

## Правило вибору шаблону

Для будь-якого нового рішення, яке має отримати процедурну надійність `deliberatio_plena`, використовується тільки `deliberation-protocol.md`.

`protocol.md` лишається як історичний або чорновий короткий формат і не повинен бути підставою для повного консенсусу.
